# FS - Football Simulator ⚽

Um jogo de futebol RPG feito em **HTML, CSS e JavaScript** — roda direto no navegador, sem instalar nada.

*A soccer RPG game made with HTML, CSS and JavaScript — runs right in your browser.*

## 🎮 Como jogar / How to play

Abra o arquivo **`index.html`** no navegador. É só isso!

*Open **`index.html`** in your browser. That's it!*

## ✨ O que tem no jogo / Features

- Criação de personagem com atributos (Futebol, Físico, Social)
- Partidas com eventos aleatórios, jogadas em sequência e skills especiais
- Sistema de XP, treino, dinheiro (K/M) e loja de skills
- Liga com 20 times, temporada, torneios continentais e mercado de transferências
- Salvar e continuar a carreira
- Bilíngue: Português e Inglês / Bilingual: Portuguese and English
- Copa do mundo
- Sistema de força dos times

## 🌍 Idiomas / Languages

Português 🇧🇷 · English 🇬🇧
